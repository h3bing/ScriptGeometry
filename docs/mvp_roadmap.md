# ScriptGeometry MVP完善需求清单

作为最小可行产品（MVP），以下是当前项目需要完善的关键项目：

## 🔴 关键阻塞项（必须修复才能编译运行）

### 1. 缺少必要的第三方库

**问题描述：**
CMakeLists.txt引用了`third_party/imgui`和`third_party/glad`，但这两个目录不存在。

**解决方案：**
```bash
# 下载ImGui
cd third_party
git clone --depth 1 https://github.com/ocornut/imgui.git
cd imgui
# 下载GLAD
mkdir -p glad/src glad/include
# 需要从glad网站生成或从vcpkg获取
```

**预计工作量：** 30分钟

### 2. 构建测试未通过

**问题描述：**
由于缺少依赖，项目无法完成完整构建验证。

**解决方案：**
- 补充依赖后执行完整构建
- 修复可能存在的编译错误
- 确保各平台兼容性

**预计工作量：** 2-4小时

---

## 🟡 功能缺陷（影响核心体验）

### 3. AI Prompt不够智能

**问题描述：**
当前AI Prompt模板可能无法生成高质量的TOON脚本。

**改进方案：**
- 提供更详细的函数签名说明
- 添加更多脚本示例到Prompt
- 实现错误脚本的自动修复建议

**代码位置：** `src/chat.cpp` - `buildDefaultSystemPrompt()`

**预计工作量：** 4-8小时

### 4. 缺少撤销/重做功能

**问题描述：**
用户操作（删除实体、修改属性等）无法撤销。

**改进方案：**
- 实现命令模式（Command Pattern）
- 维护操作历史栈
- 支持Ctrl+Z/Ctrl+Y快捷键

**预计工作量：** 8-16小时

### 5. 文件对话框使用不便

**问题描述：**
当前使用简单的输入框，用户体验差。

**改进方案：**
```cpp
// 使用原生文件对话框
// Windows: GetOpenFileName/GetSaveFileName
// Linux: gtk_file_chooser_dialog 或 zenity
// macOS: NSOpenPanel/NSSavePanel
```

**预计工作量：** 4-8小时

### 6. 缺少脚本错误反馈

**问题描述：**
脚本编译或执行失败时，用户只能看到日志，UI无明确提示。

**改进方案：**
- 在属性面板显示错误状态
- 弹出错误对话框
- 高亮错误脚本行

**预计工作量：** 4-6小时

## 结论

作为MVP，当前项目**架构完整、核心功能实现**，但存在以下关键问题需要解决：

1. **依赖缺失** - 阻止项目编译
2. **用户体验粗糙** - 影响首次使用印象
3. **AI能力待验证** - 核心卖点需要实测

建议优先解决P0和P1级别的问题，确保项目可以：
- 成功编译运行
- AI生成脚本基本可用
- 用户能完成基本的建模流程

然后可以发布v0.1版本，收集用户反馈后再进行迭代优化。
